# User Guide

## Data sources

The source connector searches for questionnaire exports in the configured project folders and, when enabled, in synchronized Google Drive locations.

## Recommended workflow

1. Export or synchronize the Google Forms responses.
2. Confirm that the file is Excel, CSV or TXT.
3. Run the application from the repository root.
4. Check the selected source and number of detected records.
5. Review generated tables and reports.
6. Archive each validated analysis with its date and software version.

## Reproducibility

For every analysis, retain:

- the software release or Git commit;
- the original data export;
- the analysis date;
- the configuration used;
- the generated outputs.

## Data protection

Use coded participant identifiers. Store identifiable information outside the public repository and follow the applicable ethics approval and institutional data-protection procedures.
