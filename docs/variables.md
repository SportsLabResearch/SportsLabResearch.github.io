# Variables

The application works with the variables already defined in the project questionnaire. It does not require changing the existing Google Form.

## Main domains

| Domain | Examples |
|---|---|
| Participant identification | coded participant identifier, date and session |
| Wellbeing | mood, stress, fatigue, pain and perceived energy |
| Recovery | sleep and perceived recovery indicators |
| Physiological monitoring | SpO₂ and blood pressure when present in the source |
| Context | treatment, session or other questionnaire-specific fields |

## Variable naming

Column names are cleaned when data are loaded, but the original questionnaire wording should remain documented in the study data dictionary.

!!! note
    The exact set of variables depends on the current Google Forms questionnaire and the columns present in each export.
