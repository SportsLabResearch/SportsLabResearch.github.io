# Quick Start

## 1. Prepare the data

Place a Google Forms export in the project input location. Supported formats include:

- `.xlsx`
- `.xls`
- `.csv`
- `.txt`

## 2. Run the application

From the project root:

```powershell
py main.py
```

## 3. Select the source

The application detects compatible questionnaire files and selects a valid source according to the project rules.

## 4. Review the outputs

Generated results are stored in the project output/results folders. Keep original questionnaire exports unchanged and work on copies when necessary.

!!! warning
    Do not upload identifiable participant data to a public repository.
