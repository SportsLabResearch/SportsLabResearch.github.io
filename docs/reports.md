# Reports

The project is designed to support clear scientific outputs without modifying the original questionnaire data.

## Expected outputs

- cleaned Excel datasets;
- descriptive summaries;
- longitudinal participant summaries;
- tables and figures;
- Word reports when enabled by the analysis workflow.

## Interpretation

Results should be interpreted together with the study protocol, the data dictionary and the clinical context. Missing values, implausible measurements and changes in questionnaire structure must be reviewed before analysis.

## Quality checklist

- Confirm the participant identifier.
- Confirm the analysis period.
- Check the number of records.
- Review missing and duplicated observations.
- Confirm units and scales.
- Record the software version used.
