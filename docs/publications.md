## Publication model

SportsLabResearch distinguishes between four types of citable output:

1. **Research articles** describing scientific findings.
2. **Validation studies** evaluating measurement or analytical performance.
3. **Technical reports** documenting architecture, verification and limitations.
4. **Software releases** preserved with version-specific citation metadata and, where appropriate, a DOI.

## Citation principles

- Cite the exact software version used in the study.
- Include the repository and archived release when available.
- State relevant parameters, configurations and processing decisions.
- Distinguish software-generated results from scientific interpretation.
- Preserve enough information for an independent researcher to reproduce the workflow.
